# EU PQC Conference Live Website SCRIPTS

[https://www.pqc-conference.eu](https://www.pqc-conference.eu)

# TLDR

1. Edit the conference edition website
  - https://github.com/pqc-symposium/eu-pqc-conference-2027
  - https://github.com/pqc-symposium/eu-pqc-conference-2025

2. Test your changes locally (see README of edition website repos)

3. Commit and push to github

4. Go to [Github Actions](https://github.com/pqc-symposium/website/actions/workflows/jekyll-gh-pages.yml) and hit `Run workflow` to build & push to the live website.
  - it will build all edition websites and combine them into the full website
